# Pong-game-for-BCI-based-on-motor-imagery
This program uses the actiCHamp aplifier, although it can be used with any device supporting LSL, and writing their corresponding settings as frequency rate, number of channels and ID of the device.

The file PONG.py is the actual pong game adapted for control by motor imagery.

The file BCIneitor.py provides visual information of the accuracy of the predictions, brain activity, and brain signal acquisition. The triggers for the creations of epoch are handled by an external code in psychopy.
